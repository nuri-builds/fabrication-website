# 🏗️ Fabrication & Engineering Website Template

A modern, fully responsive website concept designed for a fabrication and engineering services business.

## 🔗 Live Demo
Check out the live website here: [nuri-fabrication-demo.netlify.app](https://nuri-fabrication-demo.netlify.app)

## ✨ Features
* **Service Showcase:** Clean layouts detailing manufacturing, welding, and engineering capabilities.
* **Project Gallery:** Professional portfolio layout to display industrial work.
* **Contact & Inquiry Layout:** Easy-to-use forms for potential clients to request quotes.
* **Mobile-First Design:** Fully responsive layout optimized for all screen sizes.

## 🛠️ Tech Stack
* HTML5 / CSS3 / JavaScript
* Hosted on Netlify
